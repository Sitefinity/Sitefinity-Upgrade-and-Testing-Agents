# API Breaking Changes in Sitefinity CMS 15.2

> Source: [Progress Documentation](https://www.progress.com/documentation/sitefinity-cms/api-changes-in-sitefinity-cms)

## Notes

Other breaking changes:
- ControllerExtensions.GetDynamicContentType return type changed from DynamicModuleTypeВ to IDynamicModuleType
- ASP.NETВ Core: To customize the display names of enumerators, the System.ComponentModel.DescriptionAttributeВ has been replaced with Progress.Sitefinity.Renderer.Designers.EnumDisplayNameAttribute.
- ASP.NETВ Core: For enumerators, the System.ComponentModel.DescriptionAttributeВ is now used to customize the tooltip, whereas previously it was used to customize the name.
- ASP.NETВ Core: The System.ComponentModel.DescriptionAttributeВ used in enum members must be replaced with Progress.Sitefinity.Renderer.Designers.EnumDisplayNameAttribute, as this attribute is now used to customize the names. If not changed, it will likely impact customer custom widgets.
