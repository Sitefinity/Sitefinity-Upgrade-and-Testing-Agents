# API Breaking Changes in Sitefinity CMS 13.0

> Source: [Progress Documentation](https://www.progress.com/documentation/sitefinity-cms/api-changes-in-sitefinity-cms)

## Notes

IMPORTANT: After upgrade to 13.0, custom CSS styles in widget designers in the page editor might not work. For more information, see
More troubleshooting resources.
