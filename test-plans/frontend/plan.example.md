# Frontend Test Plan Example

This is an example of how to write a frontend test plan. 
Rename this file to `plan.md` and customize it for your site.

If no plan.md exists, the agent will automatically discover pages via navigation (up to 20 pages).

---

## Pages to test

Homepage
/about-us  
Contact page at /contact

## Interactions

- Test the main navigation menu dropdowns
- On the contact page, fill out the form and submit it
- Check if the search works (if there's a search box)

## Specific widgets

The homepage has a slider - make sure the arrows work
There's an accordion on the Services page - expand all items
